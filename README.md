# bqv
To manage views.
Example repository https://github.com/rerost/bqv-example

## Usage
```
## Base
bq view --dir=<DATASET_DIR> --projectid=<BQ_PROJECT_ID>

## Manage view with BQ
bqv view diff # TODO not color, not formatting
bqv view apply
bqv view dump
```
